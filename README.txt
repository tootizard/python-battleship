Uses builtin python modules, no need to install extra modules.
Built on python 3.
Tested on Windows and Linux. Should work on Macs as well.
The design folder has some documentation regarding my planning process.

To play across the internet, the host must forward port 5598 from their
router to their computer or use an application like ngrok.

To run:
    Install python 3.
    Download this code (extract it if you downloaded the tar).
    Move to your python-battleship/code/ directory in your command line interface.
    Run this command:
        python3 battleship.py
    Follow in-game prompts.
        Ctrl+C should break you out if issues arise.
